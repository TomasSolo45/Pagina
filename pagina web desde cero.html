Pasos para crear una página web completa

  1. Lo primero que debemos hacer es abrir un documento de texto en una carpeta vacia.
    2. "
  2. Lo segundo que debemos hacer es converti el documento de texto en un documento html, para esto debemos irnos a la opcion vista y luego hacer click en "Extensiones de nombre de archivo".
  3. Para este momento nos aparecera como "Documento de texto.txt" lo que debemmos hacer es dar click derecho y cambiar el nombre, puede ser el nombre que desees pero al final debes usar ".html" para que nuestro archivo se comvierta en html.
  4. Después de haber hecho todo esto debemos dar click derecho y en "Abrir con" le damos a "Bloc de Notas".
  5. Despues de esto podremos usar los diferentes codigos "html" para progamar nuestra pagina web
